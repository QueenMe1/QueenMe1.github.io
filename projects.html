Personal Projects

Security Suite

Security Suite is an ongoing learning project aimed at strengthening practical cybersecurity skills through Python development. The project is multi-phased, each focusing on different security concepts.

The first phase is a Password Strength Checker, built as a command-line python tool. It evaluates password strength based on complexity rules and uses file reading to compare user supplied passwords to against a list of commonly used passwords, helping identify weak or insecure credentials. This phase emphasizes password hygiene, file handling, and defensive input validation. View: Github and Pages
